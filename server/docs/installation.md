# Installation

1. Upload all files to your hosting account.
2. Point your web root to the `public` directory.
3. Visit `/install` in your browser.
4. Run the environment check.
5. Enter your app URL and MySQL database credentials.
6. Run migrations.
7. Create the admin account.
8. Click Finish to go to the login page.

Notes:
- Ensure the `storage` and `bootstrap/cache` folders are writable.
- Keep `APP_DEBUG` disabled in production.
